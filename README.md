# Pranajit Das - AI Engineer & Data Analyst Portfolio

A modern, high-performance personal portfolio built to showcase projects, skills, and certifications in Generative AI and Data Analytics. The design features a pristine glassmorphism aesthetic with engaging micro-animations and typography tailored for an exceptional user experience.

## ✨ Features
- **Dynamic Interface:** Custom cursor tracking, scroll progress bar, and scroll reveal animations.
- **Premium Design:** Glassmorphism translucent UI elements paired with a futuristic dark mode aesthetic.
- **Responsive Layout:** Fully optimized for seamless scaling across mobile, tablet, and desktop devices.
- **Intelligent Carousel:** Auto-typing text roles in the hero section to cleanly display expertise.

## 🛠 Tech Stack
- **Core Framework:** HTML5 & Vanilla TypeScript
- **Styling:** Tailwind CSS v4 (Using modern native setup)
- **Tooling Platform:** Vite
- **Iconography:** Lucide Icons
- **Animation Support:** ScrollReveal.js

## 🚀 Running Locally

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   cd pranajit-das-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
   *(Note: This project relies on standard Vite + Tailwind setup. You can also use yarn, pnpm, or bun)*

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
